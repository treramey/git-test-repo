# Git Steps

## Starting With Local Files

- First Time
  - `git init`
  - Make changes to files
  - Save changes
  - `git add .`
  - `git commit -m "COMMIT MESSAGE"`
  - Make Github Repo
  - `git remote add origin GITHUB_REPO_URL`
  - `git push -u origin master`
- FROM NOW ON
  - Make changes
  - Save Changes
  - `git add .`
  - `git commit -m 'COMMIT_MESSAGE'`
  - `git push`
